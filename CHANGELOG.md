# Changelog

## v0.1.0

Linc's first Windows beta includes:

- Job-detail extraction for company career pages and common job boards.
- Review notes and confidence checks for incomplete or uncertain results.
- Excel export and support for updating an existing `.xlsx` or `.xlsm` tracker.
- Duplicate handling, manual entry, browser capture, and local feedback reports.
- A portable Windows app with Python and Chromium included.
- Clearer handling for expired, blocked, and job-search pages.
