# JobLink Capture Chrome Extension

Use this extension when a job page opens in Chrome but blocks the scraper. It reads the page you already have open and sends that copy to the JobLink app running on your computer.

## Install

1. Start JobLink Beta first.
2. Open Chrome and go to `chrome://extensions`.
3. Turn on `Developer mode`.
4. Select `Load unpacked`.
5. Choose `browser_extension\joblink_capture`.

## Capture A Job

1. Open the job page.
2. Finish any human check if the site asks.
3. Click the Chrome extensions icon.
4. Select `JobLink Capture`.
5. Select `Capture this job page`.
6. Go back to JobLink and select `Load browser captures`.

Capture does not bypass a login or security check, and it can still pick up extra page text. Review the row before adding it to your tracker.
