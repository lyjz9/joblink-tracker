What JobLink handles

Reads the posting

Looks for the company, title, location, work type, salary, source, and original link.

Prepares the row

Organizes the details into the same columns used by the ready-made Excel tracker.

Shows its doubts

Marks uncertain fields for review instead of quietly filling the tracker with guesses.

From job link to tracker

  1. 1

    Paste job links

    Add links in the local app, from the command line, or through the Excel input sheet.

  2. 2

    Let JobLink read them

    It checks structured job data, common career-page layouts, and visible page text.

  3. 3

    Check the flagged rows

    Fix anything missing or suspicious, especially salary, location, and work type.

  4. 4

    Update Excel

    Download a new tracker or add the reviewed rows to your own workbook.

Where JobLink still needs help

Usually works well

  • Company career pages
  • Common ATS pages like Greenhouse, Lever, Ashby, and Workday
  • Links that do not require login or human verification

Still needs attention

  • Salary ranges and hourly pay
  • Work type detection
  • Messy locations copied from page text
  • Blocked or expired postings
